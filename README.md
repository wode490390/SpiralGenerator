## Configurations

### config.yml
```yaml
wall:
  # Block ID used to generate the wall
  material: 48
  meta: 0
ground:
  # Block ID used to generate the ground
  material: 2
  meta: 0
```

## Compiling
1. Install [Maven](https://maven.apache.org/).
2. Run `mvn clean package`. The compiled JAR can be found in the `target/` directory.
